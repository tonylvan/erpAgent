| **参数量** | ~137M |

| **能力** | embedding |
| **训练数据** | 多语言 (支持中文) |

### 量化细节

#### Q4_K_M 量化 (GGUF)

- **方法**: 4-bit 量化 + K-means 聚类
- **精度**: 接近 F16，体积接近 INT4
- **策略**: 混合精度 (关键参数保留更高精度)
- **效果**: 精度损失 <0.5%，体积减少 70%

#### F16 量化 (标准版)

- **方法**: 16-bit 半精度浮点
- **精度**: 接近原始 FP32
- **策略**: 统一精度
- **效果**: 精度损失 <0.01%，体积减少 50%
