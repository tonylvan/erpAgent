### 3. 第一次查询

**步骤**:
1. 在输入框中输入问题
2. 点击"查询"按钮或按 Enter 键
3. 等待系统处理（通常 <2 秒）
4. 查看结果

**示例问题**:
```
- "查询所有供应商"
- "显示未付款的发票"
- "供应商 A 的采购订单有哪些？"
- "上个月的销售总额是多少？"
```

## 智能问数

### 基础查询

#### 1. 简单查询

**输入**:
```
查询所有供应商
```

**输出**:
```
找到 50 个供应商：

| 供应商 ID | 供应商名称 | 类型 | 状态 |
|----------|-----------|------|------|
| 1001     | 供应商 A   | 制造商 | 活跃 |
| 1002     | 供应商 B   | 经销商 | 活跃 |
| ...      | ...       | ...  | ...  |
```

---

#### 2. 条件查询

**输入**:
```
查询状态为"未付款"的发票
```

**输出**:
```
找到 25 条未付款发票：

总计金额：¥125,000.00
平均金额：¥5,000.00
```

---

#### 3. 关联查询

**输入**:
```
查询供应商 A 的所有采购订单和发票
```

**输出**:
```
供应商 A (ID: 1001)

📦 采购订单：15 个
   - PO-2026-001: ¥50,000.00 (已完成)
   - PO-2026-002: ¥30,000.00 (进行中)
   - ...

📄 发票：20 张
   - INV-2026-001: ¥50,000.00 (已付款)
   - INV-2026-002: ¥30,000.00 (未付款)
   - ...
```

---

### 高级查询

#### 4. 时间范围查询

### 追问功能

系统支持多轮对话，理解上下文：

**第一轮**:
```
用户：查询供应商 A 的发票
系统：找到 20 张发票，总计 ¥500,000
```

**第二轮** (自动理解"这些"指代供应商 A 的发票):
```
用户：这些发票中哪些还没付款？
系统：找到 5 张未付款发票，总计 ¥125,000
```

**第三轮**:
```
用户：按金额排序
系统：已按金额降序排列...
```

---

### 查询技巧

#### ✅ 好的查询

```
✅ "查询未付款的发票"  - 清晰明确
✅ "供应商 A 的采购订单" - 具体对象
✅ "最近 30 天的销售" - 时间范围
✅ "按金额排序前 10 个" - 排序 + 限制
```

#### ❌ 模糊的查询

```
❌ "查一下那个什么" - 太模糊
❌ "所有数据" - 范围太大
❌ "看看供应商" - 想看什么？
```

## 关系图谱

### 图谱视图入门

**切换到图谱视图**:
1. 执行查询后
2. 点击顶部"图谱"标签
3. 查看节点和关系

---

### 图谱操作

#### 1. 缩放和平移

- **放大**: 鼠标滚轮向上
- **缩小**: 鼠标滚轮向下
- **平移**: 按住鼠标左键拖动
- **重置**: 双击空白区域

---

#### 2. 节点操作

**展开节点**:
- 点击节点上的"+"号
- 查看关联的节点

**收起节点**:
- 点击节点上的"-"号
- 隐藏关联节点

**查看详情**:
- 点击节点
- 右侧面板显示属性

---

#### 3. 关系高亮

**查看关系路径**:
- 点击两个节点
- 自动高亮连接路径

**查看关系类型**:
- 鼠标悬停在连线上
- 显示关系类型和属性

---

## 高级功能

### 1. 数据导出

**支持格式**:
- Excel (.xlsx)
- CSV (.csv)
- PDF (.pdf)
- JSON (.json)

**导出步骤**:
1. 执行查询
2. 点击"导出"按钮
3. 选择格式
4. 下载文件

---

### 2. 查询保存

**保存查询**:
1. 执行查询后
2. 点击"保存"按钮
3. 输入查询名称
4. 选择分类
5. 确认保存

**查看已保存**:
- 点击左侧"已保存"
- 按分类浏览
- 搜索查询名称

---

### 3. 定时报告

**创建定时报告**:
1. 进入"报告"页面
2. 点击"新建报告"
3. 选择查询
4. 设置频率（每天/每周/每月）
5. 配置接收邮箱
6. 保存

**示例**:
```
报告名称：每日销售简报
查询：今日销售订单
频率：每天早上 9:00
接收：sales@company.com
```

---

### 4. 协作分享

**分享查询**:
1. 打开已保存的查询
2. 点击"分享"按钮
3. 选择分享对象
4. 设置权限（只读/编辑）
5. 发送链接

**团队协作**:
- 创建工作区
- 邀请成员
- 分配权限
- 协作编辑

## 常见问题

### Q1: 查询结果为空？

**可能原因**:
- 查询条件过于严格
- 数据尚未同步
- 拼写错误

**解决方法**:
1. 放宽查询条件
2. 检查数据同步状态
3. 确认关键词拼写

---

### Q2: 查询速度慢？

**可能原因**:
- 查询复杂度高
- 数据量大
- 网络延迟

**优化建议**:
1. 添加时间范围限制
2. 减少返回数量
3. 使用具体字段查询

---

### Q3: 图谱显示混乱？

**解决方法**:
1. 点击"重新布局"按钮
2. 使用筛选功能减少节点
3. 展开/收起部分节点

---

### Q4: 如何查看历史查询？

**方法**:
1. 点击右上角"历史"图标
2. 按时间浏览
3. 搜索关键词
4. 点击恢复查询

---

### Q5: 数据不准确？

**排查步骤**:
1. 检查数据源连接
2. 查看同步日志
3. 确认查询条件
4. 联系管理员

---

### Q6: 忘记密码（v4.0+）？

**重置方法**:
1. 点击"忘记密码"
2. 输入注册邮箱
3. 查收重置邮件
4. 设置新密码

## 最佳实践

### 1. 查询优化

✅ **推荐**:
```
"查询 2026 年 Q1 供应商 A 的未付款发票，按金额降序，前 10 个"
```

❌ **不推荐**:
```
"查发票"
```

### 2. 结果解读

**注意**:
- 确认时间范围
- 理解聚合方式
- 检查数据完整性
- 对比历史数据

---

**大数据量查询**:
- 使用分页
- 添加索引字段
- 避免全表扫描
- 利用缓存

**注意事项**:
- 不要分享敏感查询
- 定期修改密码
- 导出文件妥善保存
