## ✅ 任务 3: JWT 认证实现

### 实现内容

**文件**: `app/api/auth.py`

**功能**:
1. 用户登录（获取 Token）
2. 用户登出
3. Token 刷新
4. 获取当前用户信息
5. 权限验证（依赖注入）

### API 端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/v4/auth/login` | POST | 用户登录 |
| `/api/v4/auth/me` | GET | 获取当前用户 |
| `/api/v4/auth/logout` | POST | 用户登出 |
| `/api/v4/auth/refresh` | POST | 刷新 Token |

### 配置参数

```env
JWT_SECRET_KEY=erp-agent-secret-2026-change-in-production
JWT_ALGORITHM=HS256
JWT_EXPIRE_MINUTES=30
```

### 测试账号

| 用户名 | 密码 | 角色 |
|--------|------|------|
| admin | admin123 | 管理员 |
| user | user123 | 普通用户 |

### 使用示例

**登录**:
```bash
curl -X POST http://localhost:8005/api/v4/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "admin", "password": "admin123"}'
```

**响应**:
```json
{
  "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "token_type": "bearer",
  "expires_in": 1800,
  "username": "admin",
  "role": "admin"
}
```

**使用 Token**:
```bash
curl http://localhost:8005/api/v4/auth/me \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```
